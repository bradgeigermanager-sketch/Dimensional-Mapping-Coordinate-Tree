# Collapse Modes

The engine supports multiple dimensional collapse modes.

## 1. Standard Collapse
User selects 3 visible axes from:
x, y, z, w, v, u, q, p, s, f

## 2. Radial Shell Collapse
(x,y,z,w) → |w|
Dims 1–3 collapse; dim 4 becomes radial distance.

## 3. Hybrid Geometry Mode
H = (|w|, Xrel, Yrel, Z_rel)
This is the default mode for:
- Tree visualization
- Measurements
- Relational analysis
